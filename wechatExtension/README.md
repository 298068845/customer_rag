# 电脑微信快捷工具

这是一个 Windows 电脑微信快捷发送工具。项目已经内置 AutoHotkey v2 运行器，不需要单独安装 AutoHotkey。

## 启动和关闭

启动：

```text
双击 start.bat
```

关闭：

```text
双击 stop.bat
```

也可以在系统托盘右键 AutoHotkey 图标，选择 `Exit`。

## 使用

- `F8`：把当前鼠标位置标记为微信输入框位置，供托盘菜单里的 `F8 定位` 模式使用。
- `~`：第一次尝试复制当前选中文字并打开编辑框；未选中文字时也会打开空编辑框。再次按 `~` 进入结果页，有文案时查询实时话术，无文案时直接进入快捷入口；结果页中继续按 `~` 依次发送勾选内容。
- `Tab`：在微信窗口中复制当前选中文字，保存到 `last-selected.txt`。
- `Esc`：关闭预览弹窗。

`~` 检索结果页支持快捷入口：

- `1` 实时话术：已接入话术 RAG 管理台的“实时话术”规则。
- `2` 领券链接、`3` 常用话术、`4` 对比图：入口已预留，暂未接入。
- `Q` 售前话术、`W` 售后话术、`E` 活动规则、`R` 自定义：入口已预留，暂未接入。
- 可点击入口按钮，也可直接按对应数字键或字母键切换。

默认使用托盘菜单里的 `UIA 定位` 模式自动定位微信输入框。需要固定点位时，先打开微信，把鼠标放到输入框里按一次 `F8`，再在右下角托盘图标的 `定位模式` 里切换到 `F8 定位`。

## 发送内容

编辑 `send-text.txt`，写入要发送的内容。

用单独一行 `---` 分隔多条消息：

```text
第一条消息
---
第二条消息
---
第三条消息
```

分条发送模式下，弹窗会为每个条目显示一个勾选框。默认全选，取消勾选的条目不会发送。

## 文件说明

- `WeChatQuickTool.ahk`：主脚本。
- `tools/autohotkey/AutoHotkey64.exe`：项目内置运行器。
- `start.bat` / `start.ps1`：启动脚本。
- `stop.bat` / `stop.ps1`：关闭脚本。
- `config.ini`：配置文件。
- `send-text.txt`：待发送内容。
